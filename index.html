Querying

cy.get()

To query for this DOM element, we write cy.get('p:nth-of-type(1)')

We can get elements by their id: cy.get('id-example')

We can get elements by their class: cy.get('class-example')

cy.contains()

We can find elements by their content cy.contains('content')

Contains finds the deepest matched DOM element containing the text.
